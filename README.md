# Modern-Porfolio
I have created a one page modern portfolio, using front-end web technologies.

# Tech Stack.
1. HTML-5
2. CSS-3
3. Bootstrap-5
4. JavaScript
5. Npm
6. Vercel

# Features.
1. Responsive for all devices.
2. Smooth transitions and animations.
3. Smooth scroll.
4. User friendly and highly accessable.
